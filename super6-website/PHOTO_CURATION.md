## SECTION 4: GAPS

These are categories that the current library does not adequately cover. If the brand is being repositioned as investor-grade premium, some of these are high priority for the next shoot.

**Critical gaps:**

1. **Female athletes — near-total absence.** One image (A2) features a female player as the primary subject. That is not defensible for a youth basketball brand in 2026. A dedicated shoot with female players in game action is needed.

2. **Dunk / above-the-rim action.** Despite the file `G1_01_Dunk_Action.jpg` existing, it is a 1200×896 downsampled AI-edited crop — no original full-resolution dunk shot exists in the library. There is no NBA-style air-under-feet action at all. This is the most common premium sports website hero image type and it's completely absent.

3. **Wide venue establishing shot.** No image shows the full court from end to end with spectators on both sides simultaneously. Everything is sideline-angle. A single overhead or wide-lens establishing shot of a Super6 event would transform the hero section options.

4. **Trophy/award ceremony close-up.** The trophy shots that exist are celebration group shots; there is no tight close-up of the trophy itself being held, inscribed, or presented — the detail/product shot that anchor an awards section.

5. **Scoreboard / game clock shot.** Not a single image captures a scoreboard showing a game in progress. This is basic tournament legitimacy signaling and it's missing.

6. **Diverse age brackets shown together.** Almost all images are one age group at a time. An image showing a 7-year-old watching a 16-year-old play (multi-generational basketball moment) would communicate the full-year program scope.

7. **Super6 staff/ownership in a professional context.** No image shows TK or the Super6 management team in a professional setting (not in the images reviewed). For investor-grade brand positioning, a staff or founder photo is needed.

8. **Multi-court wide shot.** No image shows more than one court simultaneously. If Super6 runs multi-court events, a single wide frame showing two or more courts in action would dramatically upgrade the "scale of operation" narrative.

9. **Evening/dramatic lighting.** Every image is standard fluorescent gym lighting. One image with dramatic directional lighting or a gym during a high-drama moment (final seconds, overtime atmosphere) would give the site cinematic range it currently lacks.

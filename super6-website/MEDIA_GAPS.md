## Gaps Identified

### High Priority (needed for polished launch)
1. **Location-specific venue photos** — Currently reusing lifestyle shots for Clearwater, Tampa, West Palm, and Atlanta location cards. Need at least 1 exterior + 1 interior photo per venue.
2. **Wide-format hero banner** — The feature banner section uses a video clip, but a dedicated 21:9 ratio hero image or longer-form hype video (15-30 sec) would improve the experience.
3. **Logo in SVG format** — Currently using raster PNGs. An SVG logo would be sharper at all sizes and load faster.

### Medium Priority (for enhanced content)
4. **Team/player headshots** — For an "athletes" or "testimonials" section if added later.
5. **Longer highlight video** (60-90 sec) — A sizzle reel showing game highlights, venue atmosphere, awards. The current 4-5 second clips are good for background loops but don't tell a story.
6. **Sponsor/partner logos** — If partnerships exist, logos for a "Partners" bar in the footer.

### Low Priority (future enhancements)
7. **Merchandise product shots** — If merch store is added (jerseys, basketballs, etc.)
8. **Map/illustration assets** — Custom illustrated map of Florida/Georgia with location pins.
9. **Social proof screenshots** — Social media posts, press coverage, etc.

## Current Workarounds
- Location cards for non-Orlando venues use lifestyle action shots as stand-ins
- The feature banner uses `hero-clip-3.mp4` (a 5-second basketball action loop)
- All video posters are auto-generated from first frames (manual poster images would be more polished)
- About page photo grid reuses the 25-image curated set effectively

## No TaylorMade or Third-Party Media Used
All media assets are sourced exclusively from the provided Super6 attachment folders. Zero external/stock/TaylorMade imagery was used.
